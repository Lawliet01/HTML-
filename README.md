## 术语表

有一些术语在中文中没有很好的翻译，又或者了解的人不多。为了便于读者理解，文章会使用一个大众熟知、意思相近但可能又有细微差别的概念进行替换，包括如下：

| 术语                                | 文章中实际使用的概念 |
| ----------------------------------- | -------------------- |
| User Agent(用户代理) / Implementers | 浏览器               |
| Authors                             | (网页)开发者         |
